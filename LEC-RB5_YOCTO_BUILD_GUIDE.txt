
LEC-RB5 YOCTO BUILD INSTRUCTIONS GUIDE


1. Set up a build host environment

1.1 HARDWARE REQUIREMENT

-   ​ MACHINE: x86
-   ​ OS: Ubutnu20.04 LTS (Do NOT use other versions:
    14.04/16.04/18.04/19.04)
-   ​ RAM: Minimum of 16 GB RAM
-   ​ FREE SPACE: At least 200 GB free disk space

1.2 ESSENTIAL HOST PACKAGES ARE REQUIRED TO BE INSTALLED ON YOUR HOST

    $ sudo apt-get install gawk wget git-core diffstat unzip texinfo gcc-multilib build-essential chrpath socat cpio python3 python3-pip python3-pexpect xz-utils debianutils iputils-ping python3-git python3-jinja2 libegl1-mesa libsdl1.2-dev pylint xterm zstd liblz4-tool curl git python-is-python3

  NOTE: Don’t use the root account to build Yocto. it might cause
  unpredictable errors.

1.3 “GOOGLE REPO” PACKAGES IS REQUIRED TO BE INSTALLED ON YOUR HOST

    $ mkdir ~/bin (this step may not be needed if the bin folder already exists)
    $ curl https://storage.googleapis.com/git-repo-downloads/repo > ~/bin/repo
    $ chmod a+x ~/bin/repo
    $ export PATH=${PATH}:~/bin

1.4 BEFORE STAR TO BUILD THE SOURCE, PLEASE MAKE SURE THAT GIT IS SET UP
PROPERLY WITH THE COMMANDS BELOW

    $ git config --global user.name "Your Name"
    $ git config --global user.email "Your Email"
    $ git config --list


2. Fetch the source from the git location as below

2.1 CREATE A WORKING DIRECTORY. WE’RE CREATING “LEC-RB5-YOCTO”.

    $ mkdir lec-rb5-yocto
    $ cd lec-rb5-yocto
    $ export LC_ALL="en_IN.UTF-8"
    $ export LC_CTYPE="en_IN.UTF-8"

2.2 DOWNLOAD THE REQUIRED META-LAYERS.

  manifest file to build the Yocto image :
  https://github.com/ADLINK/adlink-manifest/tree/lec-rb5-yocto-kirkstone

kirkstone branch

    $ repo init -u https://github.com/ADLINK/adlink-manifest -b lec-rb5-yocto-kirkstone -m lec-rb5-yocto-kirkstone.xml
    $ repo sync
    $ MACHINE=lec-rb5 DISTRO=rpb-wayland  . meta-adlink-qualcomm/conf/adlink-conf/adlink-rb5n-setup-release.sh

2.3 BUILD THE IMAGE

    $ bitbake rpb-weston-image
